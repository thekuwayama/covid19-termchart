`covid19-termchart` is the CLI that print "日本国内の感染者数（NHKまとめ）"  to iTerm2.

- https://www3.nhk.or.jp/news/special/coronavirus/data-all/

## Install

```bash
$ go get -u -v github.com/thekuwayama/covid19-termchart
```


## Usage

```
$ covid19-termchart
```
